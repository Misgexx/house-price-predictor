# 🏠 House Price Predictor

A beginner-friendly ML project by [Misgexx](https://github.com/Misgexx) that predicts house prices based on:
- 🛏️ Bedrooms
- 🚿 Bathrooms
- 🏗️ Year built

It includes:
- A terminal app: `predictor.py`
- A web app with Streamlit: `app.py`
